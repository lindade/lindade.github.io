About

Ich bin UX-Expertin, Produkt Designerin und Front-End Entwicklerin aus Berlin. Ich erstelle Konzepte und entwickle Software sowohl für mobile Applikation, als auch für Webseiten und Webanwendungen. Außerdem interessiere ich mich sehr für Lösungen mit AR Technologien. Besonders wichtig ist mir das Software intuitiv nutzbar ist, schön aussieht und auch Spass bei der Nutzung bereitet. Das Arbeiten in einem multinationalen Umfeld schätze ich besonders. Nicht zuletzt deshalb habe ich Teile meines Studiums in Südkorea absolviert und in internationalen Unternehmen in Spanien und Deutschland gearbeitet.

Projekte

Die Besten Nachbarn

Die Besten Nachbarn

Konzept einer Applikation zur Nachbarschaftshilfe basierend auf Nutzerbefragungen. Die Anwendung soll Menschen die gemeinsam in einer Nachbarschaften leben helfen, einfacher miteinander zu Kommunizieren und das Zusammenleben im Stadtraum vereinfachen. Ladenbetreiber sollen potenzielle Kundschaft aus der unmittelbaren Umgebung anspechen können und einfacher gefunden werden. Anbieter die normalerweise keinen Internetauftritt besitzen können Fotos hochladen und Informationen weitergeben.

Tracticle Web

Tracticle Web

Tractical ist eine Webseite eines Unternehmens mit Fokus auf einem technischen Produkt. Die Seite wurde mit HTML5, CSS3 (SASS) und JavaScript erstellt. Die Webseite ist durch den Einsatz von Bootstrap 4 vollsändig responsiv. Die Fotogallerie wurde mit dem JavaScript Plugin Ekko-lightbox für Bootstrap realisiert.

Mind Time Web

Mind Time Web

Einseitige Webseite zur Präsentation eines Buches. Die Webseite wurde in HTML und CSS (SASS) erstellt, durch den Einsatz von Bootstrap 4 ist die Seite komplett responsiv. Die Bootstrap Bibliothek wurde in diesem Projekt direkt erweitert, damit es an das gewollte Design der Seite angepasst ist. Durch Anwendung des ScrollSpy Javascript plugins wird der aktuelle Menüpunkt farblich hervorgehoben. Die dargestellten Icons werden durch die offene Fontawesome Bibliothek eingebunden.
